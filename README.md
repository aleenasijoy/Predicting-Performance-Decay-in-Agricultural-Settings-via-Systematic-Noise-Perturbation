# Predicting-Performance-Decay-in-Agricultural-Settings-via-Systematic-Noise-Perturbation
This project explores how systematic noise perturbations impact the performance of deep learning models in agricultural environments.
Deep learning models that are used to detect diseases in crops works very well with high accuracies in lab conditions, but fail miserably when implemented in real world conditions. This is due to the lab to field performance gap which can happen due to certain noises that exist in field conditions, which affects the images of the crops. Noises can be of different types, like lighting, blur, occlusion and camera angle. These images that are affected with noises are vastly different from what the model is trained on and hence the difference in performance. Because of these existent conditions, it is difficult to tell how a model will perform in realworld conditions before actually deploying it, which can can lead to waste of resources like time and cost for implementation. 
This project aims to solve this problem by creating a framework for testing which predicts how a model will perform under real-world conditions before actually implementing it in real world. Instead of finding out the model failed after implementing it in the field, we simulate the real world conditions of agriculture using artificial noise in different intensities. This helps us understand how fast the model will break and how strong or weak it is under varying niose conditions.   
Four types of noise were used in project: Gaussian Blur, change in brightness, occlusion and rotation. These noises were applied at different levels of intensities to the images from the AgriPath-LF16 dataset, focussing on corn crops which have 4 classes of diseases, namely, Grey Leaf Spot, Northern Leaf Blight, Common Rust and Healthy. Two different models were evaluated, a ResNet-50 CNN fully fine tuned for corn disease classification and AgriPath-Qwen2-VL-2B-LoRA32, which is a generalised LoRA-tuned vision-language model which was evaluated under different noises using a stratified 200 image subset to simulate low resource deployment scenario.
 
To measure the performance of these models, four methods were used. First an exponential decay curve was fitted to the results using A(I) = A₀ · exp(–kI), where A(I) is the Macro-F1 trajectories for different noise intensities, A₀ is the baseline Macro-F1, I is the intensity and k is the fragility coefficient. Secondly, this decay curve was used to find the value of k, which is the fragility coefficient, which is used to understand how fast a model would fail. Higher the value of k, the faster the model decays.  
Third, a critical threshold was calculated, which is the point where the Macro-F1 drops below 70% for CNN and 60% for LoRA model. Lastly, mean perturbation category (mPC) was used to calculate the average performance drop across all noise levels. 
The CNN achieved 97.02% baseline accuracy (Macro-F1 = 0.9703), and the LoRA model 67.0% (Macro-F1 = 0.6775). This disparity in baseline performance is primarily explained by the models' level of training specialization; the CNN was fine-tuned completely on these 4 classes of corn disease, and the LoRA model is general across all crops and all 16 classes in AgriPath-LF16. Brightness shift produced the highest fragility coefficient for both models (CNN k = 0.1469; LoRA k = 0.1513). While the CNN performs quite well under underexposure and only degrades below threshold after I=2.0 (overexposure), the LoRA model fails at both ends of the brightness spectrum. The LoRA model degenerates at intensities less than I=-0.6 and greater than I=0.6, meaning it is only effective in a small brightness range. Blur and occlusion followed the exponential model quite closely (CNN R=0.962 and 0.966, LoRA R=0.888 and 0.983, respectively), and both blur and occlusion produced the highest cumulative degradation in accuracy. Northern Leaf Blight sustained an average Macro-F1 degradation of 0.6236 under occlusion in the CNN, whereas Common Rust in the LoRA model showed the most fragility (Macro-F1 degradation under occlusion = 0.5108 and under blur=0.5254). In both the models, rotation never dropped below the critical threshold. 

In conclusion, this project presents a method in which we can predict the decay of a model before implementing it in field conditions. This helps in understanding how a model would perform under real world conditions and identify weak models, which reduces the risk of deployment. This project also helps in understanding when a model becomes unreliable in real agricultural environments.
